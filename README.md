## Назва моделі: Математичне моделювання та аналіз струмів у електричних колах.

### ПІБ та група: Власійчук Данило, АІ-235.

### Варіант: Використано режим MODE=eco (6-й номер у списку, парний). Програма успішно працює в контейнері, проводить розрахунки балансу потужності та виводить час обчислень.
